# CardinalSpace - Team Demo Script

## 🎬 4-Member Video Presentation Script

### **Team Roles & Duration**
- **Total Duration:** 10-12 minutes
- **Team Size:** 4 members
- **Format:** Screen recording + voice narration
- **Goal:** Showcase complete web app functionality
- **Preparation:** All features tested, smooth transitions ready

---

## 👥 Team Member Roles

### **Member 1: Introduction & Overview (2 minutes)**
- Welcome message and app introduction
- Landing page walkthrough
- Registration process demo
- Dashboard overview

### **Member 2: Core Features (3 minutes)**
- Real-time parking map demonstration
- Filtering and search features
- Space selection process
- Map legend and indicators

### **Member 3: Booking & Payments (3 minutes)**
- Reservation flow walkthrough
- Payment process demonstration
- Confirmation and QR code
- Booking history

### **Member 4: Advanced Features & Conclusion (3 minutes)**
- Permit management
- Vehicle registration
- Analytics and reporting
- Admin portal preview
- Closing summary

---

## 🎭 Member 1: Introduction & Overview (2 minutes)

**Speaker 1:** "Welcome to CardinalSpace! I'm [Name], and I'll be showing you our smart parking management system that's revolutionizing how organizations handle parking."

**Key Points to Cover:**
- Current parking challenges (time waste, frustration, inefficiency)
- CardinalSpace solution overview
- Three core benefits: **Efficiency**, **Convenience**, **Intelligence**

**Screen Actions:**
1. Show landing page (`/`)
2. Demonstrate portal selection
3. Highlight clean, modern design
4. Navigate to registration

**Registration & Onboarding:**
- Click "User Portal" → "Sign Up"
- Fill in registration form (demo credentials)
- Show email verification process
- **Key Point:** "Quick and painless setup - under 60 seconds!"

**Profile & Dashboard:**
- Complete profile information
- Show name update in header immediately
- Add vehicle information
- **Key Point:** "Multiple vehicle support for families"
- Display dashboard with personalized welcome
- **Key Point:** "Everything you need at a glance"

---

## 🎭 Member 2: Core Features - Real-Time Map (3 minutes)

**Speaker 2:** "Now I'll show you the heart of CardinalSpace - our real-time parking map that shows exactly what's available right now."

**Screen Actions:**

1. **Map Introduction**
   - Navigate to "Real-time parking map"
   - Show lot selection (A, B, C, D)
   - **Key Point:** "Live data updates every 10 seconds"

2. **Advanced Filtering Demo**
   - Show all available spaces first (highlight quantity)
   - **Filter by EV Charging:** "Easy to find EV charging spots"
   - **Filter by Accessibility:** "Perfect for users with special needs"
   - **Filter by Covered:** "Weather protection options"
   - **Clear Filters:** "Back to full view with one click"

3. **Space Selection**
   - Click on an available space
   - Show space details popup
   - **Key Point:** "Each space has detailed information"

4. **Visual Indicators**
   - Point out color-coded legend
   - Show special feature indicators (♿, ⚡, 🏢)
   - **Key Point:** "Intuitive visual design"

---

## 🎭 Member 3: Booking & Payments (3 minutes)

**Speaker 3:** "Let me walk you through how simple it is to reserve and pay for parking in CardinalSpace."

**Screen Actions:**

1. **Start Reservation**
   - Click "Reserve Parking" button
   - Show reservation flow steps

2. **Lot & Space Selection**
   - Select preferred lot (show pricing differences)
   - Choose specific space from filtered options
   - **Key Point:** "Price transparency helps users make informed decisions"

3. **Time & Vehicle Selection**
   - Set date and time for reservation
   - Choose from registered vehicles
   - **Key Point:** "Support for multiple vehicles per user"

4. **Payment Processing**
   - Show payment options (Credit Card, GCash, PayMaya)
   - Process demo payment
   - **Key Point:** "Multiple payment methods for convenience"

5. **Confirmation**
   - Show reservation confirmation
   - Display QR code for check-in
   - **Key Point:** "Digital tickets - no paper needed"

## 🎬 Video Production Tips

### **Recording Guidelines:**
- **Screen Resolution:** 1920x1080 (Full HD)
- **Audio Quality:** Clear microphone, no background noise
- **Smooth Transitions:** Each member hands off smoothly
- **Consistent Pace:** Follow timing guidelines closely

### **Team Coordination:**
- **Rehearse Together:** Practice handoffs and transitions
- **Backup Recording:** Record each segment separately
- **Visual Consistency:** Same browser, same settings throughout
- **Error Handling:** Have backup takes ready

### **Post-Production:**
- **Edit Together:** Smooth transitions between segments
- **Add Graphics:** Highlight key features with text overlays
- **Background Music:** Subtle, professional background track
- **Final Review:** All team members approve final version
